Dish images folder.

This folder is intended to contain menu dish images used across the Menu section.

Recommended quick action (PowerShell):

```powershell
# Copy existing gallery images into dish folder (example names)
Copy-Item .\assets\images\gallery\gallery1.jpg -Destination .\assets\images\dish\pancakes.jpg -Force
Copy-Item .\assets\images\gallery\gallery2.jpg -Destination .\assets\images\dish\omelette.jpg -Force
Copy-Item .\assets\images\gallery\gallery3.jpg -Destination .\assets\images\dish\paratha.jpg -Force
Copy-Item .\assets\images\gallery\gallery4.jpg -Destination .\assets\images\dish\butter-chicken.jpg -Force
Copy-Item .\assets\images\gallery\gallery5.jpg -Destination .\assets\images\dish\chana-masala.jpg -Force
```

You can rename files as needed to match menu items.
